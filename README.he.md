# החזר מס הכנסה — בעזרת AI

> **[English README](README.md)**

סביבת עבודה של Claude Code שמאטמטת את תהליך החזר מס ההכנסה בישראל. משכפלים את הריפו, פותחים סשן Claude, וקלוד מנווט את אתר רשות המסים בשמך באמצעות Playwright.

## איך זה עובד

1. משכפלים את הריפו ופותחים אותו ב-Claude Code
2. קלוד מנחה אותך להזין פרטים אישיים ופיננסיים
3. שרת Playwright MCP שולט בדפדפן אמיתי
4. קלוד מנווט לאתר רשות המסים ([מיסים](https://www.misim.gov.il)), ממלא את הפרטים ומגיש את בקשת ההחזר בשמך

לא נדרשת כל ידיעה בתכנות.

## דרישות מקדימות

| דרישה | פרטים |
| --- | --- |
| [Claude Code](https://claude.ai/code) | אפליקציית דסקטופ, תוסף VS Code, או CLI — `npm i -g @anthropic-ai/claude-code` |
| [Playwright MCP](https://github.com/microsoft/playwright-mcp) | שרת MCP לאוטומציה של דפדפן |
| Node.js 18+ | נדרש על ידי Playwright MCP |
| תעודת זהות ישראלית | נדרשת לאימות באתר רשות המסים |

## הגדרה

### 1. שכפול הריפו

```bash
git clone https://github.com/tupe12334/israel-tax-refund.git
cd israel-tax-refund
```

### 2. התקנת Playwright MCP

```bash
npm install -g @playwright/mcp
```

### 3. הגדרת שרת Playwright MCP ב-Claude Code

מוסיפים את הבא להגדרות MCP של Claude Code (`~/.claude/settings.json`):

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@playwright/mcp"]
    }
  }
}
```

או מריצים את פקודת ההגדרה בתוך Claude Code:

```
/mcp add playwright npx @playwright/mcp
```

### 4. פתיחת Claude Code בתיקייה

```bash
claude .
```

## שימוש

לאחר הכניסה ל-Claude Code, פשוט אומרים לקלוד:

> "עזור לי להגיש בקשה להחזר מס"

קלוד יבצע:

- שאילת שאלות לגבי הפרטים האישיים הנדרשים (מספר זהות, פרטי הכנסה, חשבון בנק להחזר)
- פתיחת דפדפן דרך Playwright
- כניסה לאתר רשות המסים עם הפרטים שלך
- מילוי והגשת טופס ההחזר
- דיווח בחזרה עם פרטי האישור

## מה כדאי להכין מראש

- מספר זהות ישראלי
- שנת המס לגביה מגישים את הבקשה
- פרטי הכנסה (תלושי שכר / טופס 106)
- מספר חשבון בנק להפקדת ההחזר
- גישה לאימייל/טלפון לאימות OTP בעת הכניסה

## פרטיות ואבטחה

- הנתונים האישיים שלך משמשים רק במהלך הסשן הפעיל ואינם נשמרים בשום מקום
- סשן הדפדפן רץ באופן מקומי על המחשב שלך דרך Playwright
- אין שליחת פרטים לשירות צד שלישי — קלוד מתקשר עם אתר הממשלה הרשמי ישירות דרך הדפדפן המקומי שלך

## פתרון בעיות

**Playwright MCP לא מתחבר**
ודא שהשרת מופיע תחת `/mcp` ב-Claude Code ומציג סטטוס ירוק. אם לא, הפעל מחדש את Claude Code לאחר עדכון `settings.json`.

**קוד OTP לא מגיע**
אתר רשות המסים שולח OTP לטלפון הרשום על תעודת הזהות. ודא שהפרטים שלך מעודכנים ב-[misim.gov.il](https://www.misim.gov.il).

**תפוגת סשן**
לאתרי ממשלה יש לרוב חלונות סשן קצרים. אם קלוד מאבד את הסשן באמצע התהליך, בקש ממנו להפעיל מחדש את הדפדפן ולנסות שוב.
